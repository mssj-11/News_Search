


### Create Project
```sh
ng new News_Search
```

### Packages
```sh
npm install
```

### Run
```sh
ng serve --o
```


##  Components
### In Components
```sh
ng g c components/search-form
```
```sh
ng g c components/news-list
```
```sh
ng g c components/navbar
```
### In Shared
```sh
ng g c shared/spinner
```
### In Services
```sh
ng g s services/news
```
```sh
ng g s services/
```
